왜 CodeTest Agent인가요?

🧠

심층 문제 이해

단순 텍스트 매칭이 아닙니다. 문제의 핵심 목표, 입력 변수, 제약 조건을 구조적으로 파악하여 정확한 가이드를 제공합니다.

⚡

반례 생성 모드

내 코드가 왜 틀렸는지 모르겠나요? 경계값, 히든 케이스, 논리적 허점을 파고드는 날카로운 반례를 제시합니다.

🗺️

단계별 솔루션

바로 정답을 알려주지 않습니다. 문제 분해부터 자료구조 선택, 알고리즘 설계까지 소크라테스식 문답으로 사고력을 키워줍니다.

🛡️

약점 분석

조건문 누락, Off-by-one 에러 등 사용자의 코드 패턴을 분석하여 자주 실수하는 약점을 짚어줍니다.

어떻게 작동하나요?

1

문제 입력

문제 텍스트나 URL을 입력하면 AI가 핵심 정보를 파싱합니다.

2

이해 검증

AI가 파악한 문제의 목표와 제약조건을 요약해 보여주고 확인받습니다.

3

모드 선택

반례를 찾을지(Counterexample), 풀이 힌트를 얻을지(Solution) 선택합니다.

4

멘토링 실행

선택한 모드에 맞춰 최적화된 피드백과 가이드를 제공합니다.

public int solution(int n) {
    // 1부터 n까지의 합을 반환
    int sum = 0;
    for (int i = 1; i < n; i++) {
        sum += i;
    }
    return sum;
}
